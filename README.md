# Portfolio-Website
A fully responsive personal portfolio website built using HTML, CSS, and JavaScript. It showcases my skills, projects, education, certifications, resume, and contact information with a modern, clean, and user-friendly design. The website includes smooth scrolling, animations, a mobile-friendly navigation menu, typing effect, project showcase, and a working contact section. Designed to demonstrate my front-end web development skills and create a strong professional online presence.
